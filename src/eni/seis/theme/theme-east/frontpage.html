CONTEXT

This project is set in the context of the Eastern Partnership Initiative that aims to deepen the EU's relations with the six Eastern Partnership Countries, supporting the promotion of environmental protection by strengthening environmental governance.

OBJECTIVE

The main objective of the project is to continue to implement the principles and practices of the Shared Environmental Information System (SEIS).
